###### **三、** **简答题** 

**1、****Servlet** **的生命周期包括哪些阶段。**

正确答案： 

Servlet 的生命周期包括加载和实例化（服务器启动或客户端首次请求时加载 Servlet 类并创建实例）、初始化（调用 init 方法）、服务（对每个请求调用 service 方法，根据请求类型可能调用 doGet、doPost 等方法）、销毁（调用 destroy 方法，通常在服务器关闭或 Servlet 长时间未使用时）。

**2、****在** **Java** **动态** **Web** **开发中过滤器（****Filter****）的应用场景有哪些？**

正确答案： 

日志记录、字符编码处理、资源访问控制、异常处理、敏感词过滤等。

**3****、****简述****vue****框架中****MVVM****模型？**

正确答案： 

MVVM 是模型 - 视图 - 视图模型的缩写，是 Vue 核心设计思想。

M（Model 模型）：指数据层，就是 Vue 中data里定义的变量、后台请求回来的数据。

V（View 视图）：指页面 DOM 结构，也就是 HTML 页面标签、展示给用户的页面。

VM（ViewModel 视图模型）：是Vue 实例，作为中间桥梁，连接 Model 和 View。

特点： Model 数据变了，View 自动更新；页面 View 中修改数据，Model 也会自动同步更新。开 发者不用手动操作 DOM，只需要维护数据，Vue 自动完成视图渲染

**4****、****简述****v-if****和****v-show****的区别？**

正确答案：

v-if 是真正的条件渲染，条件为假时直接销毁 DOM 元素；v-show 只是通过 CSS 的 display 属性 控制显示和隐藏，DOM 元素始终存在。
